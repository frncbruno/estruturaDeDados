List é só uma interface, ou seja, um “modelo” que define como uma lista deve funcionar. Ela não pode ser usada diretamente, mas serve para declarar variáveis de forma mais flexível, podendo usar diferentes tipos de listas.
ArrayList é uma lista baseada em array. Ela é rápida para acessar elementos pelo índice, mas mais lenta para inserir ou remover itens no meio. É a mais usada no dia a dia.
LinkedList é uma lista encadeada. Ela é boa para inserir e remover elementos com facilidade, mas é mais lenta para acessar posições específicas.
Vector é parecido com o ArrayList, mas é sincronizado (seguro para várias threads). Porém, é mais lento e considerado antigo, sendo pouco usado hoje.
